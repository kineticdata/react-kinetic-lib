import { Props } from 'docz';
import { FormForm } from './FormForm';

## Form

### Props

<Props of={FormForm} />

### Examples

#### Simple implementation of a `FormForm`.

```javascript
<FormForm kappSlug="services" />
```

Because no name was passed, submitting this form will create a new Form

#### Control the Layout of a `FormForm`.

```javascript
<FormForm kappSlug="services">
  {({ form }) => (
    <div>
      <h5>My Form Header</h5>
      {form}
    </div>
  )}
</FormForm>
```

Because no name was passed, submitting this form will create a new Space Attribute Definition

## Form Table

### Props

<Props of={FormTable} />

### Examples

#### Simple implementation of a `FormTable` that is a datastore.

```javascript
<FormTable columnSet={['name', 'slug']} datastore />
```

#### Control the Layout of a `FormTable`.

```javascript
<FormTable columnSet={['name', 'slug']} kappSlug="services">
  {({ table }) => (
    <div>
      <h5>My Table Header</h5>
      {table}
    </div>
  )}
</FormTable>
```
